# Instalación

- Para configurar tu proyecto, crea un entorno virtual con python -m venv .venv y actívalo ejecutando.venv\Scripts\activate (Windows) o source .venv/bin/activate (Linux/Mac).

- Después, instala las herramientas necesarias con pip install mkdocs mkdocs-material. Crea la estructura de tu documentación usando mkdocs new . y, para ver los cambios en tiempo real, ejecuta mkdocs serve y -abre en tu navegador la dirección que aparecerá en la terminal, que por lo general es:
- http://127.0.0.1:8000
